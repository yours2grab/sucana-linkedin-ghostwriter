# LinkedIn Approved Posts Database

This file contains all approved LinkedIn posts. The writer skill reads this to learn what good posts look like.

**Purpose:** Self-learning system - approved posts become examples for future posts.

---

## How to Use

When you approve a post in the writer skill:
1. The skill will ask: "Should I save this to approved posts database?"
2. If yes → post gets added here
3. Next time writer runs → it reads these approved posts as examples

---

## Format

Each approved post includes:
- Date approved
- Hook used
- Content pillar
- Post format type
- Word count
- Full post content
